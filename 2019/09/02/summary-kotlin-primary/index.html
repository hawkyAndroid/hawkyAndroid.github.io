Kotlin简介

Kotlin是一门与Swift类似的静态类型JVM语言，由JetBrains设计开发并开源。与Java相比，Kotlin的语法更简洁、更具表达性，而且提供了更多的特性，比如，高阶函数、操作符重载、字符串模板。它与Java高度可互操作，可以同时用在一个项目中。

（一）变量声明

在Kotlin中，使用 var 和 val 来声明变量。

var(可变变量)：在程序中可更改使用var关键字声明的变量的值。
val(不可变变量)：不能改变使用val关键字声明的变量的值，可简单理解为常量定义。

注意：var和val声明的变量，都可以声明时直接初始化，也可以先声明再初始化。

（二）数据类型

在Kotlin中，一切都是对象，这意味着可以在任何变量上调用成员函数和属性。
Kotlin数据类型与Java类似，主要包含以下5种数据类型。

1）数字类型

数字类型是仅包含数字类型数据变量的数据类型。它又进一步分为整数和浮点数。

2）字符类型

使用关键字Char表示字符。 char类型使用单引号(‘’)声明。

3）布尔类型

布尔数据使用Boolean类型表示。 它包含值true或false。

4）数组类型

Kotlin中的数组由Array类表示。使用库函数arrayOf()和Array()构造函数创建数组。 Array有get()，set()函数，size属性以及一些其他有用的成员函数。

{ i -> i + 1}：用于初始化并返回给定索引的数组元素的值。
val array = Array(3) { i -> i + 1 } // {1,2,3}

5）字符串类型

Kotlin中的字符串由String类表示。字符串是不可变的，这意味着无法更改String中的元素。

① 转义字符串 ：转义字符串在双引号("")内声明，并且可能包含转义字符，如'\n'，'\t'，'\b'等。
② 原始字符串 ：行字符串在三引号("""""")内声明。 它提供了在新行中声明String并包含多行的工具。行字符串不能包含任何转义字符。

【注意】

1> readLine()：输入的是String类型；Scanner(System.`in`)：可指定输入类型。
2> 在Kotlin中，不支持将较小数据类型隐式转换为较大数据类型(在java中支持)。 例如，Int不能分配转换为Long或Double。

（三）Kotlin表达式

【关键字】

is 用来判断数据类型；as：类型强制转换；
else 是一个声明；if-else：表达式结果可赋给一个变量（最后一行有效）。
break@labelname、continue@labelname：跳到指定的标签位置，其中labelname可自定义。
Kotlin 不支持任何三目运算符。

1）when 相当于Java中的switch

2）for 相当于 foreach 循环

3）Kotlin标记break和contiune

标记是标识符的形式，后跟@符号，例如abc@，label@。
break标记后跟标记名称 来终止标签标识符的主体。
continue标记后跟标记名称 来跳转到标签标识符的主体。
